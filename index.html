How to use

The easiest way to use this package is to install the package using NPM or Yarn by running the command npm i hfx-css then importing the package into your main.css file

@import './node_modules/hfx-css/dist/css/hfx.css'

To use an effect simply add hfx- then the name of the effect you want to use. Here is an example:

<button class='hfx-shake'>Hello World</button>
